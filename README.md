TinDog Practice (Bootstrap 練習)

本專案為我練習 Bootstrap 網頁切版的作業，目的在於透過模仿設計範例（「TinDog」養狗交友 App 概念）來熟悉 Bootstrap Grid System、響應式設計與元件運用。

🔍 專案概覽

專案名稱：TinDog_practice_bootstrap

使用技術：HTML + CSS + Bootstrap

檔案結構：

index.html：主首頁切版範例

css/：自訂 CSS 檔案

images/goal images/：切版與設計參考用圖片資源

專案目標：

使用 Bootstrap 的容器（container）、列（row）、欄（col）來切版。

實作響應式設計，使頁面在桌機與行動裝置上皆排版合理。

運用 Bootstrap 的元件（如按鈕、卡片、導覽列）來提升 UI 整體一致性與易用性。

練習將設計稿（或圖片資源）轉化為 HTML/CSS 架構，提高實作能力。

功能與版面亮點

頂部導覽列（Navbar）具備響應式切換。

Hero 區塊大圖 + 標題 +簡要文字介紹，提升視覺吸引力。

多欄位價格方案卡片（Pricing cards）展示不同方案選項。

客戶引用語（Testimonial）區塊，模擬真實用戶回饋。

響應式圖片、欄位，在手機尺寸可正常顯示。
